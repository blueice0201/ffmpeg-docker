Supported compose fonts (auto-downloaded on first use when `COMPOSE_FONT_AUTO_DOWNLOAD=true`):

- `NotoSansSC-Regular.otf` — 思源黑体（简体）
- `NotoSansTC-Regular.otf` — 思源黑体（繁体）
- `NotoSerifSC-Regular.otf` — 思源宋体（简体）

Select fonts in the Web UI, or set `fontId` in manifest `style`. Override with `COMPOSE_FONT_PATH` for a single global font.
